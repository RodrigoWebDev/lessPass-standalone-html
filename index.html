About

This is a lightweight, standalone HTML version of the Less Pass password manager. In this version, you can download the single HTML file from this page and have your favorite password manager with you forever, without depending on the website, repository, internet connection and without the need to install any app on your phone. Your system only needs a modern browser. You can also back up this file wherever you want.

How to use this version

It's simple. Just download this page in this link and open in your favorite browser(I recommend chrome but other browsers should work)
